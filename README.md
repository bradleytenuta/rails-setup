### Versions

ubuntu - 18.04
ruby - 2.5.9
rails - 4.2.8
bundler - 1.16.0

In Gemfile use:
`gem 'sass-rails', '5.0.3'`
`gem 'rails', '4.2.8'`
`gem 'web-console', '2.0.0'`


Deploying to Heroku works but make sure you have git set up and pushed first.

Inclcude `ruby '2.5.9'` at the top of gemfile to use correct version on Heroku.
